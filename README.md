# Overview of the Project

The Micro Marketplace is a simple text-based Python project that allows users to buy and sell items within a small marketplace system.
It is entirely console-driven and does not require any external libraries or frontend.

This project demonstrates core Python concepts like:

Lists

Dictionaries

Functions

Loops

User input

Menu-driven programming

It is designed for beginners who want to build a functional project using only basic Python.
